# Re-Chalk
Bindings of [chalk](https://github.com/chalk/chalk) for [reasonml](https://reasonml.github.io/)
# What's missing?
As you can imagine, there are some things missing. Very few, but still. Here's the list:
* RGB
* HSL
* HWB
* HSV
* Hex

# Build
```
npm run build
```

# Build + Watch

```
npm run start
```
